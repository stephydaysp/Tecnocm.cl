# tecno_cmc — Retro Magia Medieval (Opción 2)

Estructura de proyecto lista para abrir en navegador.

## Estructura
```
/index.html
/css/style.css
/js/script.js
/assets/   (opcional para imágenes/sonidos)
```

## Uso
1. Abre `index.html` en tu navegador.
2. (Opcional) Edita colores en `css/style.css` (`:root`).
3. Ajusta el haz de luz en `js/script.js` (`MAX_POINTS`, `EASE`).

## Créditos de imágenes (demostración)
Imágenes referenciadas desde Unsplash (vínculos directos en el HTML).
